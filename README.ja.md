# ckan-es

[CKAN](https://ckan.org/) にインスパイアされ、DenoとVanilla JavaScriptで構築されたサンプルデータカタログサイトです。このプロジェクトは、オープンデータセットの閲覧とフィルタリングを行うための軽量なフロントエンドのデモとして機能します。

## ライブデモ

**[https://ckan-es.vercel.app/](https://ckan-es.vercel.app/)**

このデモでは、データカタログの典型的なインターフェースを確認できます。ユーザーはデータセットのリストを表示し、組織やフォーマットなどのカテゴリでフィルタリングを行い、クリックして各データセットの詳細情報を閲覧することができます。

## 主な機能

- **データセットの検索:** ローカルのCSVファイルから読み込んだオープンデータセットのカタログを閲覧できます。
- **詳細ビュー:** 各データセットのすべてのメタデータを確認できる専用ページにアクセスできます。
- **クライアントサイド・フィルタリング:** 組織、グループ、タグ、フォーマットでデータを即座にフィルタリングします。
- **シンプルなアーキテクチャ:** 静的ファイル配信用の最小限のDenoサーバーを使用し、すべてのロジックをフロントエンドで処理します。

## はじめに

### 前提条件

システムに [Deno](https://deno.land/) がインストールされていることを確認してください。

### サーバーの起動

1. リポジトリをクローンします。
2. プロジェクトのルートディレクトリからサーバーを実行します:
    ```bash
    deno run -A ckan.js
    ```
3. ブラウザで `http://localhost:8081` を開きます。

## データソース

このプロジェクトでは、静的なサンプルファイル `static/datasets.csv` をデータソースとして使用しています。

データの取得と操作のロジックはすべて、`static/index.html` 内の非同期関数 `api()` を用いてクライアントサイドで処理されます。実際のAPIに接続するには、目的のエンドポイントからデータを取得するようにこの関数を変更する必要があります。
