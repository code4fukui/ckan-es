# ckan-es

Deno を使ったデータカタログサイトのサンプルプロジェクトです。

## デモ
オープンデータの検索や閲覧ができます。
[デモサイト](https://ckan-es.vercel.app/)

## 機能
- オープンデータのカタログ表示
- データセットの詳細表示
- データフィルタリング

## 使い方
```bsh
deno run -A ckan.js
```

## データ・API
このプロジェクトではサンプルデータを使用しています。  
実際のオープンデータAPIを利用する場合は、`ckan.js`内の`api()`関数を適宜変更する必要があります。
